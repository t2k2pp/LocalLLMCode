
# 本ガイドラインに従い、業務に役立つミニアプリを作成します。

miniapp.mdの中に記載されているミニアプリを作成しましょう。作成するアプリはprojects配下に格納します。projectsフォルダや、現在のフォルダに格納されているファイルの取扱いについては後述のファイル一覧を確認してください。

## ファイル一覧

- README_MANY.md
	- 本資料です
- miniapp.md
	- 作成対象のアプリ情報が記載されています
- todo.md
	- 実施中のタスク、および完了したタスクの情報をこのファイルに記載していってください。
- projects フォルダ
	- この中にminiapp.mdに書かれている番号に対応するフォルダを作成し、その中にファイルを作成してください。
	- 作成するファイルはREADME.md（作成するアプリの概要、起動・利用手順）、ソースファイル、requirements.txt（作成するアプリで必要なライブラリ、pip install のパラメータとして指定できる形式）
